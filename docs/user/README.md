## Onderhoud

Werk bij een functionele wijziging zowel de betrokken workflowdocumentatie onder `docs/ai` als de relevante gebruikershandleiding bij. Een handleiding beschrijft alleen gedrag dat in de applicatie en de eigenaar-documentatie is vastgelegd.

Zichtbare menu-items blijven afhankelijk van:

- actieve modules;
- rolrechten;
- persoonlijke rechten;
- land-, team- en gebruikersscope;
- lifecyclestatus van het record.
